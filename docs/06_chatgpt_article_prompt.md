# ChatGPT用 記事生成プロンプト

## 1. 目的

管理画面に入力した素材をChatGPTに貼り付け、以下をまとめて作成するためのプロンプト。

- Webサイト用活動レポート本文
- 一覧用抜粋文
- Instagram用キャプション
- ハッシュタグ
- 記事タイトル整形案

## 2. 基本プロンプト

```text
あなたはJAしまね雲南地区本部の講座サイト「うんなん暮らしの学び帖」の記事作成担当です。
以下の素材をもとに、Webサイト用の記事文とInstagram用の投稿文を作成してください。

サイトの方向性：
- 暮らしの雑誌風
- 地域ニュースサイト風
- やわらかい
- 読みやすい
- スマホで読みやすい
- 募集感よりも活動レポート感を大切にする

Web記事の条件：
- 暮らしの読み物風
- 硬すぎず、軽すぎない
- 300〜500字程度
- 絵文字は使わない
- 講師名、内容、参加者の雰囲気を自然に入れる
- 写真が少なくても成立する文章にする
- 個人が特定されすぎる表現は避ける
- 赤ちゃん・子ども・参加者の写真掲載には配慮した表現にする

Instagram文の条件：
- Web記事より短め
- 少し親しみやすく
- 絵文字は控えめにしてもよい
- 写真と一緒に投稿しやすい文にする
- 最後に自然なハッシュタグをつける

出力してほしいもの：
1. 記事タイトル案
2. 一覧用抜粋文
3. Web記事本文
4. 参加者の声の整形
5. Instagram用キャプション
6. ハッシュタグ

素材：
講座名：
開催日：
タイトル：
講師名：
講師肩書き：
講師Instagram：
内容メモ：
メニュー・制作物・体験内容：
参加者の声：
募集情報：
写真メモ：
その他補足：
```

## 6. 参加者の声 整形用プロンプト

```text

- 個人が特定される表現は避ける
- 代表引用として使えるものを1つ選ぶ
- その他は吹き出しカード向けに短く整える

【吹き出しカード】

・...
・...
```

## 6-2. 写真相談用プロンプト（現行プロトタイプで追加）

管理画面の写真管理から、記事情報（講座名・開催日・記事タイトル）と写真枚数を差し込んでコピーできる、写真の掲載可否・役割・トリミングを相談するためのプロンプト。AIの回答は貼り付け欄で受け取り、自動反映はせず、写真ごとの設定に手動で反映する。

```text
以下の写真を、JAしまね雲南地区本部の「うんなん暮らしの学び帖」に掲載する前提で確認してください。

サイトの雰囲気：
生成り背景、細い線、ミニマル、やさしい暮らしの記録。
派手すぎず、少し明るく、彩度控えめ、あたたかい雰囲気。

記事情報：
講座名：{{講座名}}
開催日：{{開催日}}
記事タイトル：{{タイトル}}

確認してほしいこと：
1. メイン写真に向いている写真
2. ギャラリー向きの写真
3. 使わない方がよい写真
4. 顔や個人情報が写りすぎていないか
5. トリミングするなら、中央・上寄せ・下寄せ・左寄せ・右寄せのどれがよいか
6. 各写真の短いキャプション
7. 記事全体として写真が足りているか

出力形式：
写真1：
- 掲載判断：
- 役割：
- トリミング位置：
- 注意点：
- キャプション：

写真2：
- 掲載判断：
- 役割：
- トリミング位置：
- 注意点：
- キャプション：
```

## 7. 例：華凜 発酵食品

### 入力例

```text
講座名：華凜
開催日：令和8年1月27日
タイトル：からだ想いのやさしい食事 手軽に簡単！発酵食品の魅力
講師名：飯塚生美子 先生
講師肩書き：島根県家の光講師
内容メモ：味噌やヨーグルト、塩麹など、身近な発酵食品を使った料理教室。参加者同士で協力して調理した。
メニュー：
・味噌とヨーグルトで和風炊き込みご飯
・ヨーグルト味噌汁
・ラーパーツァイ
・鶏肉の塩麹焼き 蒸し焼き野菜添え
・混ぜるだけちぎりパン
参加者の声：
・野菜と発酵食品を使っての簡単料理、ヨーグルトの新しい利用法はすごく勉強になりました。
・とても美味しくいただきました。
・いつも思いますが、品数の多い料理をみんなで手際良く作るので豪華な食事になり幸せな気分になります。
```

### 出力例

```text
【記事タイトル】
華凜｜からだ想いのやさしい食事 手軽に簡単！発酵食品の魅力

【一覧用抜粋文】
味噌やヨーグルト、塩麹など、身近な発酵食品を使った料理教室を開催しました。

【Web記事本文】
この日は、発酵食品をテーマにした料理教室を開催しました。

講師には、島根県家の光講師の飯塚生美子先生をお迎えし、味噌やヨーグルト、塩麹など、身近な発酵食品を使った料理を学びました。

参加者同士で声をかけ合いながら調理を進め、会場にはあたたかな雰囲気が広がりました。完成した料理を囲みながら、家庭でも試してみたい食材の使い方や、発酵食品の魅力について話が弾む時間となりました。

【今回の内容】
・味噌とヨーグルトで和風炊き込みご飯
・ヨーグルト味噌汁
・ラーパーツァイ
・鶏肉の塩麹焼き 蒸し焼き野菜添え
・混ぜるだけちぎりパン

【参加者の声】
代表引用：
「ヨーグルトの新しい利用法がすごく勉強になりました。」

その他：
・とても美味しくいただきました。
・みんなで手際よく作るので、豪華な食事になり幸せな気分になります。

【Instagram用キャプション】
華凜 活動レポート🌿

今回は、発酵食品をテーマにした料理教室を開催しました。
味噌、ヨーグルト、塩麹など、身近な食材を使って、からだにやさしいメニューに挑戦。

参加者のみなさんからは、ヨーグルトの新しい使い方が勉強になったという声もいただきました。

【ハッシュタグ】
#JAしまね #雲南 #華凜 #女性大学院 #料理教室 #発酵食品 #家の光
```

## 8. 注意事項

- 参加者個人が特定される表現は避ける
- 赤ちゃんや子どもが関係する講座では、写真や個人情報に配慮した表現にする
- 講師名や肩書きは、入力された内容から変えない
- 不明な内容を勝手に補わない
- JAらしい信頼感は残す
- ただし硬すぎる行政文書のようにはしない
